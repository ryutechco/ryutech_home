--- layout: default title: ECU รีแมพ section_id: remapping lang: th services: - - title: ECU คืออะไร? icon: hdd desc: ในรถยุคปัจจุบัน เครื่องยนต์ถูกควบคุมด้วยระบบไฟฟ้าโดยใช้ระบบอิเลกทรอนิกส์ที่ใช้ชื่อว่า Engine Control Unit (ECU) หรือ กล่องควบคุมเครื่องยนต์ โดย ECU มีหน้าที่ควบคุมอุปกรณ์ทุกอย่างที่เกี่ยวข้องกับเครื่องยนต์ เฃ่น ปั้มน้ำมัน พัดลมหม้อน้ำ หัวฉีด หัวเทียน และ อื่นๆ โดยโปรแกรมและกระบวนการควบคุมนั้นจะถูกโปรแกรมอยู่ในECU ตัวกล่องECU อาจจะอยู่ด้านในรถ หรือในห้องเครื่องรถก็ได้

ในการผลิตรถยนต์มาตรฐานมีผู้ผลิตECUหลายรายเช่น Bosch Siemens Denso และ อื่นๆ แต่ในรถยนต์ที่ใช้ในการแข่งขัน จำเป็นจะต้องแก้ไขการควบคุมอุปกรณ์ของรถยนต์ได้ทุกอย่างซึ่งไม่สามารถทำได้โดยใช้กล่องควบคุมจากโรงงาน ในส่วนนี้จะเป็นกล่องควบคุมเฉพาะทางเช่น MoTec Haltech และ อื่นๆ

color: img: ecu.png - title: อะไรคือการรีแมพ ECU? icon: chart-area desc: การโปรแกรมหรือรีแมพคือกระบวนการที่แก้ไขข้อมูลที่อ่านมาจากECU ระบบข้อมูลการควบคุมรถยนต์จากโรงงานที่อยู่ในECUจะถูกอ่านออกมาโดยใช้อุปกรณ์เฉพาะทาง หลังจากนั้นทางเราจะทำการปรับค่าตัวแปรที่ควบคุมเครื่องยนต์เช่น แรงดันน้ำมัน ปริมาณการจ่ายน้ำมัน แรงดันอากาศเข้าเครื่องยนต์(บูสท์) การจุดระเบิด และระบบคันเร่ง การเปลี่ยนค่าตัวแปรเหล่านี้ทำให้ความประพฤติของเครื่องยนต์ ประสิทธิภาพ และ สมรรถณะ เปลี่ยนไป

โปรแกรมควบคุมเดิมจากโรงงานถูกพัฒนามาด้วยความคำนึงถึงผลกระทบจากหลายอย่างเช่น การขับขี่ในสถานที่ทั่วโลก ค่าความปลอดภัยเพิ่มเติมของอุปกรณ์ และประเทศที่จำหน่ายรถยนต์ ในการที่จะอยู่ภายใต้ข้อกำหนดทุกอย่าง ค่าตัวแปรของการควบคุมเครื่องยนต์นั้นจะถุกลดลงมาให้น้อยกว่าลิมิทของเครื่องยนต์ค่อนข้างมาก ระยะห่างระหว่างโปรแกรมเดิมกับลิมิทนั้นคือสิทธิภาพที่ถูกซ่อนอยู่ที่การรีแมพECUจะสามารถดึงออกมาได้

color: img: remapping.jpg - - title: Chip Tuning icon: microchip desc: The term ‘Chip Tuning’ refers back to 1980s where there are no On Board Diagnostics (OBD) port to diagnose data in the early model of ECUs. Back then, reprogramming or remapping the ECU are done by directly accessing the physical memory components of the ECU. This means that the ECU must be taken out of the vehicle, disassembled and actually take off the memory chip from the ECU board.

Nowadays, OBD port is equipped as a standard in every vehicle and can be used to access the data of the ECU without requiring physical access to the ECU. However, there are some models that OBD port cannot be used to reprogram the ECU. These models requires direct access to the ECU, but without removing any ECU components.

color: orange img: chip_tuning.jpg - title: OBD Remapping icon: plug desc: Vehicles manufactured since the year 2000 are equipped with OBD-II port as a mandatory standard specification. With this OBD-II port, there are protocols to read and write the memory contents of the ECU without accessing the actual hardware. Reprogramming or remapping via OBD-II port is the most common way to remap the vehicle today.

Without having to remove any hardware from the vehicle, remapping is done easier, safer, and quicker. Remapping tool connects securely to the OBD-II port to read and write the maps data of the ECU.
